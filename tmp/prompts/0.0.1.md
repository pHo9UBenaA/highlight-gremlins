以下の要件を統合したvscode拡張機能を作成してほしい

# 要件

## 1. 全角スペースとtrailing spacesの強調表示と削除
- ファイル上: 全角スペース強調表示
- 全角スペース -> 半角スペースへ自動置換するコマンドパレットから実行できるコマンド

## 2. trailing spacesの強調表示と削除
- ファイル上: trailing spaces強調表示
- trailing spaces -> 全てのtrailing spacesを削除するコマンドパレットから実行できるコマンド

# 3. Gremlinsの強調表示
- 以下のgremlinsが含まれる行の行番号を強調表示
